Chrome Extension Settings

Jobmaster

Everything from the Python settings page now lives here: profile, resume, recruiter answers, candidate source data, summary generation, cover-letter template, and imports/exports.

Profile

Core candidate details the extension uses everywhere.

Resume

Store a browser-owned resume file for autofill and keep helpful resume notes.

Recruiter Answers

Fallback JSON answer bank for questions not covered by structured profile fields. Entries support `question`, `aliases`, `answerType`, `answer`, `aiHint`, and `platformHints`.

Autofill Settings

Control confidence thresholds, optional AI fallback plumbing, and per-platform adapter enablement.

Candidate Source Data

Store every useful signal: GitHub, LinkedIn notes, resume bullets, education, project highlights, preferences, and recruiter context.

LaTeX Cover Letter Template

Stored as LaTeX and rendered against the current job plus profile fields.

Tracker Snapshot

Quick counts from the extension-local job tracker.
